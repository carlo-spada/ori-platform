# Task: Stripe Dashboard Configuration

## Objective
Configure the necessary products and prices in the Stripe Dashboard to support the "Plus" and "Premium" subscription tiers with monthly and yearly billing options.

## Scope
This task involves manual configuration within the Stripe Dashboard. Claude should provide clear instructions for a human operator to perform these steps.

## Acceptance Criteria
*   Two Products created in Stripe: "Plus" and "Premium".
*   For each Product, two Prices created:
    *   Monthly: $5 for Plus, $10 for Premium.
    *   Yearly: $48 for Plus, $96 for Premium.
*   A clear, concise set of instructions for a human to follow to set this up.
